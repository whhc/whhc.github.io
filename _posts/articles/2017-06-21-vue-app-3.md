### Vue.resource应用&header组件

* 安装`vue-resource`:`npm install --save vue-resource`

* 在`main.js`中引入并使用:

```javascript
import VueResource from 'vue-resource'

Vue.use(VueResource)
```

* 在`App.vue`中引入`seller`数据;

```javascript
export default {
  data(){
    return {
      seller: {}
    }
  }
}
```

每个vue组件都有一个生命周期,`header`组件需要在创建时就请求数据,则:

```javascript
  created () {
    this.$http.get('/api/seller').then((response) => {
      response = response.body
      if (response.errno === 0) {
        this.seller = response.data
      }
    })
  }
```
> 推荐`const ERR_NO = 0`然后判断`response.errno`是否等于`ERR_NO`以便维护;

* 在`<v-header></v-header>`中绑定`seller`:

```html
`<v-header v-bind:seller="seller"></v-header>`
```
在`header`组件中通过`props`接受传递的属性:

```javascript
export default {
  props: {
    seller: {
      type: Object
    }
  }
}
```

### 组件样式

两点:

* 通过判断dpi引入不同的图片:

`mixin.styl`:

```css
bg-image($url)
  background-image: url($url + "@2x.png")
  @media (-webkit-min-device-pixel-radio: 3),(min-device-pixel-radio: 3)
    background-image: url($url + "@3x.png")
```

* 通过`classMap`引入不同的图片:

`header.vue`:

script:

```javascript
export default {
    props: {
      seller: {
        type: Object
      }
    },
    created () {
      this.classMap = ['decrease', 'discount', 'special', 'invoice', 'guarantee']
    }
  }
```

template:

```html
<span class="icon" :class="classMap[seller.supports[0].type]"></span>
```
style(stylus):

```css
&.decrease
  bg-image('decrease_1')
&.discount
  bg-image('discount_1')
&.guarantee
  bg-iamge('guarantee_1')
&.invoice
  bg-image('invoice_1')
&.special
  bg-image('special_1')
```
